Desarrollador Frontend Senior

Tech Solutions S.A.- Remoto

Descripción del puesto

Estamos buscando un Desarrollador Frontend Senior con experiencia en React y TypeScript para unirse a nuestro equipo dinámico. El candidato ideal tendrá un fuerte enfoque en la experiencia del usuario y habilidades para trabajar en un entorno ágil.

Responsabilidades

  • Desarrollar y mantener aplicaciones web utilizando React y TypeScript.
  • Colaborar con diseñadores y desarrolladores backend para crear soluciones integrales.
  • Optimizar aplicaciones para máxima velocidad y escalabilidad.

Requisitos

  • Mínimo 5 años de experiencia en desarrollo frontend.
  • Experiencia sólida con React, TypeScript, HTML5, CSS3 y JavaScript ES6+.
  • Conocimiento de herramientas de control de versiones como Git.

Acerca de la empresa

Tech Solutions S.A. es una empresa líder en tecnología que se especializa en soluciones innovadoras para clientes globales. Valoramos la diversidad, la creatividad y el crecimiento profesional de nuestros empleados.
